# 更新日志

## v2.6.0 (企业级高可用版) - 2024-01-22

### 🎉 重大更新

本版本是系统架构的重大升级,引入了企业级高可用架构、完善的备份恢复机制、全面的安全加密、以及生产级的监控和运维体系。

### ✨ 新增功能

#### 1. 备份与灾难恢复系统
**文件**: `utilities/backup_manager.py` (NEW)

- 📦 **自动备份调度**
  - 支持每日、每周、每月定时备份
  - 可配置备份保留策略
  - 自动清理过期备份

- 💾 **多种备份类型**
  - 配置文件备份
  - 数据库备份(SQLite、MySQL、PostgreSQL)
  - 日志文件备份
  - 交易数据备份

- 🔄 **灾难恢复**
  - 一键恢复到指定备份点
  - 支持部分恢复(仅恢复指定组件)
  - 恢复前自动验证备份完整性

- ☁️ **远程备份支持**
  - S3兼容对象存储(AWS S3、MinIO、阿里云OSS)

### 🔄 迁移指南

从v2.5升级到v2.6:

1. **安装新依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **创建配置文件**
   ```bash
   # 所有配置文件都有默认值,可以直接启动
   # 如需自定义,复制示例配置文件并修改
   cp config/backup_config.json.example config/backup_config.json
   ```

3. **运行系统自检**
   ```bash
   python utilities/system_checker.py
   ```

4. **启动系统**
   ```bash
   # 单服务器模式
   ./start_system.sh

   # 双服务器模式
   # 主服务器
   python utilities/server_role_manager.py server1
   ./start_system.sh

   # 备份服务器
   python utilities/server_role_manager.py server2
   ./start_system.sh
   ```
